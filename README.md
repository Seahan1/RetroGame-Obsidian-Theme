# RetroGame

RetroGame is an Obsidian theme inspired by classic handheld gaming hardware, especially the original Game Boy aesthetic: cool gray shells, muted cartridge-label colors, tactile controls, and a soft retro UI atmosphere.

It is designeåd to make Obsidian feel more like a collectible retro device while still staying practical for long-term writing, note organization, and daily use.

## Highlights

- **Game Boy-inspired retro palette** with cool gray surfaces and muted accent colors
- **Consistent light-style appearance** across both light and dark system modes
- **Tactile UI styling** with raised buttons, pressed states, and handheld-like controls
- **Cartridge-style accents** in navigation and functional UI areas
- **Pixel-inspired title styling** for stronger retro identity
- **Polished plugin surfaces** for metadata panels, Kanban views, and workspace modules

## Design direction

RetroGame is built around a few core ideas:

1. **Retro handheld shell**  
   The main interface uses deeper gray-beige tones inspired by classic portable consoles.

2. **Cartridge color accents**  
   Dusty red, yellow, green, teal, blue, and violet are used as layered highlights to create distinction without overwhelming the workspace.

3. **Physical button feel**  
   Tabs, buttons, and controls use shadows and pressed interactions to feel more like real handheld hardware.

4. **Readable everyday UI**  
   The theme keeps its nostalgic visual language while staying usable for reading, editing, and managing notes.

## What is styled

RetroGame focuses strongly on interface polish across the Obsidian workspace, including:

- Sidebars and file explorer
- Tabs and top header controls
- Buttons, toggles, inputs, and search UI
- Metadata / Properties panels
- Kanban plugin cards and lanes
- Callouts, tables, lists, links, and code blocks

## Best fit

RetroGame is a good fit if you want:

- a retro gaming mood without excessive saturation
- a more memorable and collectible-looking Obsidian setup
- a light interface with stronger personality
- a theme that emphasizes UI character as much as content presentation

## Installation

### Manual install

1. Copy the `RetroGame` folder into your vault’s `.obsidian/themes/` directory.
2. Open **Settings → Appearance** in Obsidian.
3. Select **RetroGame** from the theme list.

## Notes

- RetroGame is intentionally tuned to behave like a **light visual system in both light and dark modes**.
- The centered inline note title uses **Pixelify Sans** for a stronger retro display feel.
- The theme puts special focus on UI modules, controls, and plugin surfaces rather than turning the writing area into a heavily stylized novelty layout.
