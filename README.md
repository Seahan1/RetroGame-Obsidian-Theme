# RetroGame

RetroGame is an Obsidian theme inspired by classic handheld gaming hardware, especially the original Game Boy aesthetic: cool gray shells, muted cartridge-label colors, tactile controls, and a soft retro UI atmosphere.

It is designeåd to make Obsidian feel more like a collectible retro device while still staying practical for long-term writing, note organization, and daily use.

## Highlights

- **Game Boy-inspired retro palette** with cool gray surfaces and muted accent colors
- **Consistent light-style appearance** across both light and dark system modes
- **Tactile UI styling** with raised buttons, pressed states, and handheld-like controls
- **Cartridge-style accents** in navigation and functional UI areas
- **Pixel-inspired title styling** for stronger retro identity
- **Polished plugin surfaces** for metadata panels, Kanban views, and workspace modules
